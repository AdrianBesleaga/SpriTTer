# SpriTTer
Twitter Clone made with Spring FrameWork in Java using REST Services with AJAX jQuery

✓Register
✓Login
✓Post a message
✓Load message list
✓Search for a specific user name
✓Add user to friend list


[http://localhost:8080/twitter/](http://localhost:8080/twitter/)
